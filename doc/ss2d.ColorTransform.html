Class: ColorTransform

ss2d. ColorTransform

new ss2d.ColorTransform(redMultiplier, greenMultiplier, blueMultiplier, alphaMultiplier, redOffset, greenOffset, blueOffset, alphaOffset)

ColorTransform.js, line 40

可使用 ColorTransform 类调整显示对象的颜色值。可以将颜色调整或颜色转换应用于所有四种通道：红色、绿色、蓝色和 Alpha 透明度。

Name Type Description
redMultiplier

与红色通道值相乘的十进制值。
greenMultiplier

与绿色通道值相乘的十进制值。
blueMultiplier

与蓝色通道值相乘的十进制值。
alphaMultiplier

与 Alpha 透明度通道值相乘的十进制值。
redOffset

255 到 255 之间的数字，加到红色通道值和 redMultiplier 值的乘积上。
greenOffset

255 到 255 之间的数字，加到绿色通道值和 greenMultiplier 值的乘积上。
blueOffset

255 到 255 之间的数字，加到蓝色通道值和 blueMultiplier 值的乘积上。
alphaOffset

255 到 255 之间的数字，加到 Alpha 透明度通道值和 alphaMultiplier 值的乘积上。
